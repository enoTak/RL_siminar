Bellman方程式の解法(2)

強化学習でのタスク

  • 価値関数v(s∣π)v(s|\pi)v(s∣π)を最大化するような方策π\piπを求める

  • 環境モデルのMarkov Dicision Process p(s′,r∣s,a)p(s', r | s, a)p(s′,r∣s,a)が与えられた場合は動的計画法によってπ\piπを求めることができる

    • Policy iteration
    • Value iteration

環境モデルが使えない場合

  • 環境モデルが明らかでない
  • 環境モデルが大きすぎて直接計算が難しいときは?
    • モンテカルロ法
    • TD学習 (Temporal Difference)

モンテカルロ法

強化学習でのモンテカルロ法

  • vπ(s)=E(Gt∣St=s)v_{\pi}(s) = E(G_t|S_t = s)vπ​(s)=E(Gt​∣St​=s)を計算したい
    • 期待値計算にモンテカルロを使う!
  • エピソードを一つ作ると, その途中から始めたパスも一つのエピソード
    ⇒これも利用する
    • 各初期状態sssに対しての条件付期待値を計算したい
    • ので、途中経過も利用できる
    • 状態が多すぎると必要なエピソード多くなる?

アルゴリズムイメージ

  1. エピソード {St,Rt∣t=0,1,⋯ ,T}\{S_t, R_t |t=0, 1, \cdots, T \}{St​,Rt​∣t=0,1,⋯,T}を1つ生成する
  2. 各時刻tttでのGt=∑l=1γl−1Rt+lG_t = \sum_{l=1} \gamma^{l-1}R_{t+l}Gt​=∑l=1​γl−1Rt+l​を計算
  3. 時刻t+1t+1t+1において (t≥0t \geq 0t≥0)
    Vt+1(s)=1Nt+1(s)∑k=0tGkχ(Sk=s)Nt+1(s)=∑k=0tχ(Sk=s)\begin{aligned} V_{t+1}(s) &= \frac{1}{N_{t+1}(s)} \sum_{k=0}^{t} G_k \chi(S_k = s) \\ N_{t+1}(s) &= \sum_{k=0}^{t} \chi(S_k = s) \end{aligned}Vt+1​(s)Nt+1​(s)​=Nt+1​(s)1​k=0∑t​Gk​χ(Sk​=s)=k=0∑t​χ(Sk​=s)​を計算
  4. 次のエピソードへ?

漸化式で書き換えると

Vt+1(s)=Vt(s)+1Nt+1(s)(Gt−Vt(St))χ(Sk=s)Nt+1(s)=Nt(s)+χ(St=s)\begin{aligned} V_{t+1}(s) &= V_t(s) + \frac{1}{N_{t+1}(s)} (G_t - V_t(S_t)) \chi(S_k = s) \\ N_{t+1}(s) &= N_t(s) + \chi(S_t = s) \end{aligned}Vt+1​(s)Nt+1​(s)​=Vt​(s)+Nt+1​(s)1​(Gt​−Vt​(St​))χ(Sk​=s)=Nt​(s)+χ(St​=s)​

  • これを元に, 1Nt+1(s)\frac{1}{N_{t+1}(s)}Nt+1​(s)1​のところをαt\alpha_tαt​とした漸化式を使ってもOK
    • Robbins-Monro条件を満たす限りは収束性が保証される
    • 実用的には小さな定数αt=α\alpha_t = \alphaαt​=αでもOK

モンテカルロ法の欠点

  • オンライン学習できない
    • GtG_tGt​の計算にエピソード終点まで用いるから
  • 各エピソードが有限の長さ(エピソディック)なMDPでないといけない
  • バイアスは小さいがバリアンスが大きい

TD学習でこれをクリアできる

TD学習

アイデア

  • Bellman方程式
    vπ(s)=∑aπ(a∣s)∑s′∑rp(s′,r∣s,a)(r+γvπ(s′))v_{\pi}(s) = \sum_{a} \pi(a|s) \sum_{s'}\sum_{r}p(s', r | s, a) (r+\gamma v_{\pi}(s'))vπ​(s)=a∑​π(a∣s)s′∑​r∑​p(s′,r∣s,a)(r+γvπ​(s′))
  • ここの r+γvπ(s′)r+\gamma v_{\pi}(s')r+γvπ​(s′) は現状態sssにおいての次の状態s′s's′に対するvπv_{\pi}vπ​の推定値と見なせそう
    (Gt=Rt+1+γGt+1G_t = R_{t+1} + \gamma G_{t+1}Gt​=Rt+1​+γGt+1​に注意!!)
  • Gt≈Rt+1+γVt(St+1)G_t \approx R_{t+1} + \gamma V_t(S_{t+1})Gt​≈Rt+1​+γVt​(St+1​)としてモンテカルロ更新式を書き直す

TD学習の更新式

  • 更新式

    Vt+1(s)=Vt(s)+α(Rt+1+γVt(St+1)−Vt(St))χ(Sk=s)\begin{aligned} V_{t+1}(s) &= V_t(s) + \\ &\alpha (R_{t+1} + \gamma V_t(S_{t+1}) - V_t(S_t)) \chi(S_k = s) \end{aligned}Vt+1​(s)​=Vt​(s)+α(Rt+1​+γVt​(St+1​)−Vt​(St​))χ(Sk​=s)​

  • TD誤差δt+1\delta_{t+1}δt+1​を用いると

    δt+1=Rt+1+γVt(St+1)−Vt(St)Vt+1(s)=Vt(s)+αδt+1χ(St=s)\begin{aligned} \delta_{t+1} &= R_{t+1} + \gamma V_t(S_{t+1}) - V_t(S_t) \\ V_{t+1}(s) &= V_t(s) + \alpha \delta_{t+1} \chi(S_t = s) \end{aligned}δt+1​Vt+1​(s)​=Rt+1​+γVt​(St+1​)−Vt​(St​)=Vt​(s)+αδt+1​χ(St​=s)​

  • 時刻tttで分かっている情報(VtV_tVt​, StS_tSt​, St+1S_{t+1}St+1​, Rt+1R_{t+1}Rt+1​)だけで計算可. つまりオンライン化ができる

TD学習の欠点

  • 1ステップ先の情報だけで更新してしまう
    • バリアンスは小さくなる
    • 代わりにバイアスが大きくなる

nnn-ステップTD学習/TD(λ\lambdaλ)法で解消したい

nnn-ステップTD学習とTD(λ\lambdaλ)法

アイデア
